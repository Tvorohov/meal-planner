# Планувальник меню (Meal Planner)

Інтерактивний веб-застосунок для планування сніданків, обідів і вечер на кілька тижнів.
Додавайте страви в слоти через випадаючий список (зручно на телефоні), ведіть каталог
страв. Один користувач, local-first, без бекенду (MVP).

**Демо:** https://tvorohov.github.io/meal-planner/

## Стек

Vite · React · TypeScript · Tailwind CSS · zustand (persist) · lucide-react.

## Запуск

```bash
npm install
npm run dev      # дев-сервер (Vite)
npm run build    # збірка в dist/
npm run preview  # перегляд збірки
```

Стан зберігається в `localStorage` (ключ `meal-planner-uk`), тому план переживає
перезавантаження сторінки.


## Взаємодія

Страви додаються в слот через випадаючий список `+ Додати…` (показує лише страви,
що підходять для цього прийому їжі). Прибрати страву — кнопкою «×» на картці. Каталог
страв (додати / редагувати / видалити) — у бічній панелі, на телефоні — в нижньому листі
за кнопкою «Страви».

## Прийоми їжі

Сніданок і вечеря — щодня. Обід додається лише на вихідні (Сб, Нд). Страви для вечері
доступні і для обіду.

## Правила попереджень

У `src/validation.ts`, кожне правило — окрема чиста функція. Перевірки йдуть по всій
стрічці тижнів, включно зі стиком між тижнями. Щоб вимкнути правило — приберіть його з
масиву `RULES`.

- Чіт-вечері в сусідні дні.
- Одна й та сама страва в сусідні дні (в одному прийомі їжі).
- Рибні сніданки в сусідні дні.
- Дитяча страва в слоті без другої страви для дорослих.

## Деплой

Пуш у гілку `main` автоматично запускає GitHub Actions
([`.github/workflows/deploy.yml`](.github/workflows/deploy.yml)): `npm ci` → `npm run
build` → публікація на GitHub Pages. У продакшені Vite `base` дорівнює `/meal-planner/`.

## Критерії приймання

- У слоті «Сніданок» список `+ Додати…` показує лише сніданкові страви; «Шакшуку» можна додати.
- Список слота вечері не містить суто сніданкових страв.
- Дитячу й дорослу страву можна додати в один слот.
- Нова страва додається з форми й одразу з'являється у списках відповідних слотів.
- Піца в суботу + бургери в неділю → в обох горить попередження.
- Перезавантаження сторінки зберігає план.
- Розкладка придатна для телефона.

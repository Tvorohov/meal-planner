# Планировщик меню (Meal Planner)

Интерактивное веб-приложение для планирования завтраков и ужинов на несколько недель.
Перетаскивайте блюда из бэклога в сетку недель, перемещайте их между днями и приёмами
пищи, ведите каталог блюд. Один пользователь, local-first, без бэкенда (MVP).

## Стек

Vite · React · TypeScript · Tailwind CSS · zustand (persist) · @dnd-kit · lucide-react.

## Запуск

```bash
npm install
npm run dev      # дев-сервер (Vite)
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

Состояние сохраняется в `localStorage` (ключ `meal-planner-v1`), поэтому план переживает
перезагрузку страницы.

## Правила предупреждений

В `src/validation.ts`, каждое правило — отдельная чистая функция. Проверки идут по всей
ленте недель, включая стык между неделями. Чтобы отключить правило — уберите его из
массива `RULES`.

- Чит-ужины в соседние дни.
- Одно и то же блюдо в соседние дни (в одном приёме пищи).
- Рыбные завтраки в соседние дни.

## Критерии приёмки

- «Шакшуку» можно перетащить из бэклога в «Завтрак» понедельника — она там появляется.
- Блюдо «только ужин» нельзя бросить в слот завтрака.
- Блюдо можно перетащить с ужина вторника на ужин четверга — оно перемещается.
- Детское и взрослое блюдо помещаются в один слот ужина.
- Новое блюдо «Плов» (ужин / говядина) добавляется из формы и появляется в бэклоге.
- Пицца в субботу + бургеры в воскресенье → у обоих горит предупреждение.
- Перезагрузка страницы сохраняет план.
- Экспорт даёт JSON-файл, импорт восстанавливает то же состояние.
- Раскладка пригодна для телефона.

Полная спецификация: [`docs/MEAL_PLANNER_SPEC.md`](docs/MEAL_PLANNER_SPEC.md).
